#  개요
- express, socket.io를 사용한 실시간 채팅, 실시간 그림 그리기 구현
- canvas api를 사용하여 그림 그리기 구현
- 주어진 제시어를 그려서 맞추는 게임 / 게임 시작 시 랜덤으로 출제자, 제시어 지정

# 사용한 기술
- Front-end : `JavaScript` `babel` `gulp` `socket.io`
- Back-end : `express` `pug`
- Deploy : `heroku`
